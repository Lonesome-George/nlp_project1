迭代训练选择语料过程：
1.训练初步模型：利用50条语料训练出一个简单模型；
2.选择语料：利用分类器模型对所有语料进行判断，选出最无法确定情感极性的m条语料；
3.人工标注：对选出来的m条语料选取特征明显的进行人工标注；
4.调整模型：将新标注的语料加入训练集中，调整模型的参数；
5.如果选择出来的语料没有达到200条，则返回2，否则结束。

选语料的策略：
每次选30条，看到特征不明显或者相似的就删除，直至选够200条。


采用《多方法融合的微博情感分析》的建模方法如下：
使用空间向量模型表示文档，首先会提取微博的特征，然后将微博映射到 N 维特征空间中的一个特征向量，最后使用 SVM 在特征空间中进行分类。主要分为三个步骤：数据预处理，特征词提取，权重计算和分类。数据预处理主要任务是删除微博中的噪声数据，特征词提取主要任务是使用特征向量表征文档，权重计算和分类主要任务是计算特征向量的权重并且进行情感分类。
1.数据预处理：预处理包括三个步骤：数据清洗。
2.特征词提取：分词和去停用词，采用改进的CHI算法提取特征词。
3.权重计算和分类：采用TF-IDF算法计算特征词的权重，训练模型并分类。

【任务说明】
预处理的任务：
去除微博中的噪声数据，这些噪声数据主要包括：回复，话题，引用等。
特征词提取的任务：
对于每个类（我们这里有两类1和-1），获取CHI得分最高的N个词，然后将所有的类的特征词求并集，得到的集合就构成了特征集合。
权重计算的任务：
将每条文本包含的特征词使用tf*idf的方法计算其权重。
也就是说，特征词提取是确定使用哪些词作为特征；而权重计算则是计算每条文本中每个特征词的权重，然后生成该文本对应的特征向量。两者是不相交的。


代码规范：
1.命名尽量使用英文小写，多个单词之间使用下划线；

输入输出说明：
1.预处理
input:	[id	label	text](不包含中括号，下同)
output:	[id	label	pped_text]
2.特征词提取
input:	[id	label	pped_text]
output:	[id	label word,freq;word,freq...]
3.权重计算和分类
input:	[id	label word,weight;word,weight]
output:	[id	label vector]
4.模型训练
input:	[id	label vector]
output:	model
