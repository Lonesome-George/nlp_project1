IMPORTANT!
To Predict:
1.运行extract_features.py抽取特征词，特征词的数目可以在代码中修改；
2.运行weight.py计算特征词的权重；
3.运行main_pred.py预测文本极性。
每次预测新文本集时只需运行main_pred.py，表示使用现有的特征词训练模型并预测；
如果要使用新的特征词训练模型，只要从第一步开始做即可。

To Choose Corpora:
1.Remove all files under the directory "Training/ChosenSet";
2.Run extract_features.py to extract feature words, you can set in code how many words to choose;
3.Rum weight.py to compute the weights of feature words;
4.Run main_select_corpus.py, and you can get 50 weibo texts under the above directory;
5.Label the chosen texts, and go back to 2.
